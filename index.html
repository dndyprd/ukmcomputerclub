Tentang Kami

Tentang Kami

UKM Computer Club Politeknik Negeri Bali adalah organisasi mahasiswa yang berfokus pada pengembangan keterampilan di bidang teknologi dan kreativitas. UKM Computer Club memiliki 4 divisi berbeda yaitu divisi Office, Visual Communication Design, Web Development, dan juga Game Development yang dimana masing-masing divisi memiliki visi misi nya tersendiri. UKM Computer Club sendiri memiliki berbagai program kerja unggulan seperti CCDT, PNB Eagles serta IntechFest.

Divisi

Divisi

Office

Latihan untuk mengoprasikan aplikasi perkatoran Microsoft Office seperti Word dan excel secara interaktif dan lebih .

Web Dev

Belajar membuat sebuah website dari tampilan hingga ke sistem menggunakan framework utility-first CSS yaitu Tailwind CSS.

Game Dev

Tidak hanya bermain game saja, tapi disini kita juga belajar untuk membuat sebuah game itu sendiri dengan engine yang ada.

VCD

Eksplorasi dunia desain kreatifitas dengan teknologi yang ada, belajar membuat ilustrasi dan desain keren.
